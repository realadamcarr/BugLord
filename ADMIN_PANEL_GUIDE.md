# Admin Panel Usage Guide

## How to Access the Admin Panel

The admin panel is a hidden feature designed for testing purposes. To access it:

### Method 1: From the Explore Tab (Character Screen)
1. Go to the **Explore** tab 
2. In the "Level Progress" section, **tap the level text (e.g., "Level 1") 5 times quickly**
3. The admin panel will appear

### Method 2: From the Notes Tab (Home Screen)  
1. Go to the **Notes** tab (Home screen)
2. In the progress header, **tap the level text 5 times quickly**
3. The admin panel will appear

## Admin Panel Features

### 🚀 Set Max Level (25)
- Instantly sets your level to 25 (higher than the highest cosmetic unlock at level 20)
- Unlocks all cosmetics immediately
- Perfect for testing the full cosmetics showcase

### Custom Level Setting
- Enter any level between 1-50 in the text input
- Useful for testing specific unlock levels
- Example: Set level 12 to test the "Superhero Cape" unlock

### XP Boost Buttons
- **+100 XP**: Small XP boost for minor level progression
- **+500 XP**: Medium XP boost 
- **+1000 XP**: Large XP boost for quick level ups
- These will automatically handle level ups if you gain enough XP

### 🔄 Reset Progress
- Resets all progress back to Level 1 with 0 XP
- Useful for testing the progression system from the beginning
- **Warning**: This will clear all your progress!

## Testing Scenarios

### Test All Cosmetics
1. Use "Set Max Level (25)" to unlock everything
2. Go to the Cosmetics tab to see all unlocked items
3. Check that the character displays the highest tier cosmetics

### Test Level Progression
1. Use "Reset Progress" to start from Level 1
2. Use the XP boost buttons to gradually level up
3. Watch for level up notifications and cosmetic unlocks

### Test Specific Unlocks
1. Set custom level to test specific cosmetic unlocks:
   - Level 2: Baseball Cap
   - Level 5: Crown  
   - Level 10: Wizard Hat
   - Level 15: Party Hat
   - Level 20: Trophy (highest unlock)

## Tips
- The admin panel automatically syncs between tabs (notes and character screen will show the same level)
- Level up notifications will still appear when using XP boosts
- The admin panel is modal - tap the X or outside to close it
- Your admin changes persist until you manually reset or change them again

## Security Note
This admin panel is intended for development and testing only. In a production app, this would typically be removed or protected behind authentication.
