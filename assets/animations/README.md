# Animation Assets

Place Lottie `.json` files here. Free animations available at https://lottiefiles.com

## Required files

| File | Used in | Suggested search |
|------|---------|-----------------|
| `capture-success.json` | `CaptureSuccessOverlay` | "bug catch", "success sparkle", "star burst" |
| `xp-levelup.json` | Future: XP level-up screen | "level up", "confetti", "stars" |

## Missing file behaviour

Both `LottieEffect` and `CaptureSuccessOverlay` fail **silently** — the app
works normally, the Lottie layer just doesn't render until you add the asset.

- https://lottiefiles.com/animations/confetti (confetti burst)
- https://lottiefiles.com/animations/stars (star pop)
